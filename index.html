Hello This is Git Tutorial

git --version => check version of grid

git config --global user.name <name> => set user name of git

git config --global user.email <email> => set user email of git

git config --global init.defaultBranch <name> => set name of branch master to what we want

git status => check all file we have in directory that no track (no git add)

git add <file> => add file to git and after we use git add we need to commit it (can use git add --A/--all for add all the file)

git commit -m <message> => commit file with message

git log => see commit code, history(date) and author who commit (press q to exit)

git checkout <commitCode> => switch version of code/project

git checkout main / git checkout -f main => switch version back to latest version

git branch -M main => to change master branch to main

git remote add origin <github> => to change master branch to main

git push -u origin main => after git branch -M main and git remote add origin <github> we need to push local to remote repo

git branch <branch-name> => create new branch

git checkout <rbanch-name> => switch branch to branch-name

git checkout -b feature-branch-name => create new branch and switch branch to it

git push --set-upstream origin branch-name => publish local branch to remote repo
